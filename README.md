# projectives
This is a Python package for the automatic segmentation and analysis of projective techniques. Particularly, in its current state, it allows AI-augmented analysis of the 12-Circles Drawing Technique.

The journal article detailing the development and performance of the AI-augmented analysis will be linked when the article is finished.
